## Architecture and netcode

| Requirement ID              | Governing behavior                                                                         | Executable evidence                                                                              |
| --------------------------- | ------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------ |
| `IMPL-4.2-FIXED-STEP`       | Simulation and scheduler advance bounded fixed ticks.                                      | `packages/server/src/scheduler.test.ts`; shared simulation suites                                |
| `IMPL-4.4-RECONCILIATION`   | Local input applies immediately; acked input drops once; remaining input replays in order. | `packages/client/src/net/local-prediction.test.ts`; `tests/network/prediction-adversity.test.ts` |
| `IMPL-4.4-CORRECTIONS`      | Small corrections smooth, large corrections snap, authoritative state remains intact.      | `packages/client/src/net/local-prediction.test.ts`                                               |
| `IMPL-4.5-CLOCK-SYNC`       | Lowest-RTT NTP-lite samples estimate server offset.                                        | `packages/client/src/net/clock-sync.test.ts`; `packages/client/src/net/net-client.test.ts`       |
| `IMPL-4.7-INTERPOLATION`    | Remote planes/projectiles interpolate; late/duplicate/underrun snapshots follow policy.    | `packages/client/src/net/snapshot-buffer.test.ts`                                                |
| `IMPL-5.3-PURE-SEEDED-SIM`  | State progression uses explicit config/dt/RNG and reproduces seeded behavior.              | shared simulation and RNG suites; `tests/network/prediction-adversity.test.ts`                   |
| `IMPL-7.4-INPUT-VALIDATION` | Boundary rejects malformed/non-finite input and clamps axes; stale sequences drop.         | `packages/shared/src/protocol.test.ts`; `packages/server/src/match.test.ts`                      |
| `IMPL-7.5-TEARDOWN`         | Sockets, waiting rooms, and matches release on leave/shutdown.                             | server matchmaker/match tests; `tests/integration/matchmaking.test.ts`                           |
| `IMPL-8.1-SERVER-PHASES`    | Clients mirror server phase/result and never authoritatively advance matches.              | `packages/client/src/net/net-client.test.ts`; browser lifecycle journeys                         |
| `IMPL-12-ASSET-LICENSES`    | Referenced glTF/audio exist, validate, stay within weight bounds, and have credits.        | `tests/assets/assets.test.ts`                                                                    |
